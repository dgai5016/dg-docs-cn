# Dataview 的伙伴们

下面是一些可能与 Dataview 相关工作流搭配使用的插件：

 - [MetaEdit](https://github.com/chhoumann/MetaEdit) - 轻松添加或更新 YAML 属性和 Dataview 字段

此外，还有一些插件借助 Dataview 来完成其功能所需的部分「重活」，列表同样非详尽：

 - [Kanban](https://github.com/mgmeyers/obsidian-kanban) - 在 Obsidian 中创建以 Markdown 为底层的看板
 - [Breadcrumbs](http://publish.obsidian.md/breadcrumbs-docs) - 让你能够在 Obsidian Vault 中可视化自定义构建的层级结构
 - [Supercharged Links](https://github.com/mdelobelle/obsidian_supercharged_links) - 允许你基于笔记元数据对 Vault 中的链接进行样式定制

完整的列表可以通过 GitHub 的 [Dependents](https://github.com/blacksmithgu/obsidian-dataview/network/dependents) 功能查看。
