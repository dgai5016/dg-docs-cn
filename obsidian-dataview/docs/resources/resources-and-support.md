# 其他资源

Dataview 会相当频繁地推出新功能和修复，因此请注意这些资源可能略微滞后于最新版本。
欢迎直接为这份清单、文档贡献内容，或者联系原始资源的作者请求更新。

### GitHub Discussions

GitHub 仓库有一个相当活跃的 [Discussions 页面](https://github.com/blacksmithgu/obsidian-dataview/discussions/)，里面有数十个已被回答的问题。

### Obsidian 论坛

[Obsidian 论坛](https://forum.obsidian.md/)上有大量问答以及其他有趣的小贴士。
遇到问题时不妨先在论坛搜索一下答案，尤其是看起来像是新手问题时。

### Discord

[Obsidian Members Group Discord 服务器](https://obsidian.md/community)中有一个 `#dataview` 频道。
同样地，你的问题大概率已经被问过了，所以请先尝试在频道里搜索——当然，Discord 的搜索功能时灵时不灵，这是众所周知的。
如果你没找到满意的答案，这里是你能找到的最接近实时支持的地方，但不保证有人会立即回复。
这里有非常多热心的人，所以不必怯于提问。

## 支持

遇到问题应该去哪儿求助？以下是我们建议的尝试顺序。

1. 先在 GitHub Discussions 和 Obsidian 论坛里搜索你的问题。
2. 再在 Discord 中搜索可能的解决方案。
3. 根据你问题的复杂程度：
  - 如果你需要接近同步的沟通，使用 Discord。请注意我们是一个由志愿者组成的社区，回复可能会有延迟。
  - 如果你的问题可能需要数天时间、并且更适合异步沟通，那就到 GitHub 开一个 Discussion。
4. 如果你发现了一个 bug，请到[仓库的 issues](https://github.com/blacksmithgu/obsidian-dataview/issues) 中提交报告。
